# StoryForge 사용자 요구 사양

> 최종 업데이트: 2026-03-19

## 브라우저 호환성

### 권장 브라우저

| 브라우저 | 최소 버전 | 비고 |
|----------|-----------|------|
| Chrome | 90+ | 권장 (개발/테스트 기준) |
| Edge | 90+ | Chromium 기반 |
| Firefox | 90+ | |
| Safari | 15+ | |
| Samsung Internet | 15+ | |

### 지원 불가

| 브라우저 | 이유 |
|----------|------|
| Internet Explorer | ES Module, CSS 변수, `inset` 등 미지원 |
| Chrome 89 이하 | CSS `inset` 속성 미지원 |
| Safari 14 이하 | CSS `gap` in flexbox 미지원 |

## 사용 기술별 브라우저 요구사항

| 기술 | 필요 이유 | 최소 지원 |
|------|-----------|-----------|
| CSS Custom Properties (`var()`) | 테마 시스템 (다크모드) | Chrome 49, FF 31, Safari 9.1 |
| CSS `inset` shorthand | 오버레이/모달 위치 | Chrome 87, FF 66, Safari 14.1 |
| CSS `gap` in flexbox | 레이아웃 간격 | Chrome 84, FF 63, Safari 14.1 |
| ES2020+ (`?.`, `??`) | 옵셔널 체이닝, nullish 병합 | Chrome 80, FF 72, Safari 13.1 |
| `localStorage` | 다크모드 설정 저장 | 모든 현대 브라우저 |
| SVG `<marker>` | 관계도 화살표 | 모든 현대 브라우저 |
| `data-theme` attribute selector | 다크모드 CSS 전환 | 모든 현대 브라우저 |
| `HashRouter` | GitHub Pages SPA 라우팅 | 모든 현대 브라우저 |

## 기기/화면

| 항목 | 권장 | 최소 |
|------|------|------|
| 화면 너비 | 1024px 이상 (데스크톱) | 360px (모바일) |
| 해상도 | 1920x1080 | 360x640 |

### 반응형 대응 현황

| 화면 크기 | 대응 수준 |
|-----------|-----------|
| 1024px+ (데스크톱) | 완전 대응 |
| 768px~1024px (태블릿) | 대시보드 사이드바 숨김, 기본 대응 |
| ~768px (모바일) | 탭 가로 스크롤, 집필 사이드바 숨김, 세계관 세로 전환 |
| ~640px (소형 모바일) | 그리드 1열, 폼 세로 배치 |

## 네트워크/성능

| 항목 | 내용 |
|------|------|
| 인터넷 연결 | 초기 로드 시 필요 (SPA 정적 파일) |
| 빌드 결과물 | CSS ~58KB(gzip 9KB) + JS ~306KB(gzip 92KB) |
| 오프라인 | 초기 로드 후에는 동작 가능 (데이터는 메모리/샘플) |
| 서버 의존성 | 없음 (프론트엔드 전용, 백엔드 API 없음) |

## 현재 제한사항

| 항목 | 설명 |
|------|------|
| 데이터 저장 | 서버/DB 없음, 새로고침 시 샘플 데이터로 초기화 |
| 인증 | 샘플 계정만 지원 (`demo@storyforge.com` / `demo1234`) |
| 이미지 업로드 | 미지원 (커버 이미지 URL 직접 입력 가능) |
| 내보내기 | 미지원 (작품 데이터 파일 내보내기/가져오기) |
| 다중 사용자 | 미지원 (싱글 유저 전용) |
