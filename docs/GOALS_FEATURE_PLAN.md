# 목표 설정 & 달성도 기능 기획

> 작성일: 2026-03-19
> 상태: 기획 초안

---

## 1. 기능 개요

작가가 창작 활동의 **목표를 설정하고 달성도를 시각적으로 확인**할 수 있는 기능.
꾸준한 집필 습관을 형성하고 성취감을 제공하여 창작 동기를 유지하는 것이 목적.

---

## 2. 목표 유형

### 2-1. 일일 목표 (Daily Goals)
| 항목 | 설명 |
|------|------|
| 글자 수 목표 | 하루에 작성할 글자 수 (예: 1,000자) |
| 집필 시간 목표 | 하루에 에디터에서 보낸 시간 (예: 30분) |
| 집필 여부 | 오늘 한 번이라도 글을 썼는지 (Y/N) |

### 2-2. 작품 목표 (Work Goals)
| 항목 | 설명 |
|------|------|
| 전체 글자 수 | 작품의 목표 분량 (예: 100,000자) |
| 챕터 수 | 목표 챕터 수 (예: 30챕터) |
| 완결 목표일 | 작품 완결을 목표로 하는 날짜 |

### 2-3. 기간 목표 (Period Goals)
| 항목 | 설명 |
|------|------|
| 주간 목표 | 이번 주 작성할 총 글자 수 |
| 월간 목표 | 이번 달 작성할 총 글자 수 |
| 연속 집필 | N일 연속 집필 목표 (스트릭) |

---

## 3. 달성도 시각화

### 3-1. 대시보드 통합
- **오늘의 목표** 카드: 일일 글자 수 진행 바 + 퍼센트
- **연속 집필** 배지: 현재 스트릭 일수 + 최고 기록
- **주간 히트맵**: 7일간 집필 현황 (GitHub 잔디 스타일)

### 3-2. 작품별 목표 패널
- 작품 상세 > 설정(⚙) 내에 "목표" 섹션 추가
- 프로그레스 바: 현재 글자 수 / 목표 글자 수
- 챕터 진행률: 완성 챕터 / 목표 챕터
- 남은 일수 & 일평균 필요 글자 수 자동 계산

### 3-3. 통계 페이지 (향후)
- 일별/주별/월별 글자 수 차트
- 작품별 성장 그래프
- 가장 생산적인 시간대 분석

---

## 4. UI 위치 & 플로우

```
대시보드
├── [통계 카드 영역] 오늘의 목표 진행 카드 추가
├── [내 작품] 각 작품 카드에 진행률 미니 바 표시
└── [최근 활동] 목표 달성 시 축하 메시지 표시

작품 상세
├── 상단바: 오늘의 글자 수 / 목표 표시
└── 설정(⚙): 목표 설정 섹션
    ├── 일일 글자 수 목표
    ├── 전체 글자 수 목표
    ├── 목표 챕터 수
    └── 완결 목표일

프로필
└── 달성 기록 / 스트릭 히스토리
```

## 5. 데이터 구조 (초안)

```typescript
interface DailyGoal {
  wordCount: number       // 일일 글자 수 목표
  writingMinutes: number  // 일일 집필 시간 목표 (분)
  enabled: boolean
}

interface WorkGoal {
  targetWordCount: number | null    // 작품 목표 글자 수
  targetChapters: number | null     // 목표 챕터 수
  deadline: string | null           // 완결 목표일 (YYYY-MM-DD)
}

interface DailyLog {
  date: string          // YYYY-MM-DD
  wordsWritten: number
  minutesSpent: number
  worksEdited: string[] // 편집한 작품 ID 목록
}

interface GoalStats {
  currentStreak: number    // 현재 연속 집필 일수
  longestStreak: number    // 최장 연속 집필 일수
  weeklyWords: number[]    // 최근 7일 글자 수
  monthlyWords: number     // 이번 달 글자 수
}
```

## 6. 구현 우선순위

| 단계 | 기능 | 복잡도 |
|------|------|--------|
| **1단계** | 작품별 목표 설정 UI (설정 패널 내) | 낮음 |
| **1단계** | 대시보드 오늘의 목표 카드 | 낮음 |
| **2단계** | 연속 집필 스트릭 표시 | 중간 |
| **2단계** | 작품 카드에 진행률 바 | 낮음 |
| **3단계** | 주간 히트맵 | 중간 |
| **3단계** | 통계 차트 페이지 | 높음 |

---

## 7. 고려사항

- **데이터 지속성**: 현재 프론트엔드 전용이므로 localStorage 활용 → 백엔드 연동 시 마이그레이션
- **부담 최소화**: 목표 설정은 선택사항. 미설정 시에도 자연스럽게 통계만 표시
- **동기 부여 UX**: 달성 시 축하 애니메이션, 미달성 시 압박감 주지 않기
- **작품별 독립**: 각 작품마다 별도 목표 설정 가능
