## 구현 완료 기능

### 기본 인프라
- [x] 랜딩 페이지, 로그인/회원가입 (샘플 인증)
- [x] 대시보드 (사이드바 + 작품 카드 캐러셀 + 최근 활동)
- [x] 내 작품 페이지 (그리드/리스트 뷰 전환)
- [x] 새 작품 생성 (장르 콤보박스)
- [x] 프로필 페이지
- [x] 다크모드 토글 (사이드바 하단, localStorage 저장)

### 작품 상세 (탭 구성: 집필 → 챕터 → 캐릭터 → 관계도 → 세계관 → 용어집)

**집필 탭**
- [x] 챕터 사이드바 + 에디터 레이아웃
- [x] 챕터 제목 인라인 편집, 글자 수 카운트
- [x] 자동 저장 설정 (간격 선택)
- [x] 용어집 핀 고정 용어 하이라이트 + 클릭 시 설명 툴팁

**챕터 탭**
- [x] 챕터 목록 (역순), 새 챕터 추가 폼
- [x] 챕터 클릭 → 집필 탭으로 이동

**캐릭터 탭**
- [x] 아코디언 레이아웃 (여러 캐릭터 동시 펼침/편집)
- [x] 이름, 역할, 나이, 성별, 외형, 성격, 설명, 배경, 태그
- [x] 신규 캐릭터 하단 추가 + 자동 펼침

**관계도 탭**
- [x] SVG 원형 배치 다이어그램
- [x] 양방향 관계: 평행 직선 2줄 + 화살표 방향 표시
- [x] 라벨: 양방향 시 from 쪽 1/3 지점에 분산 배치
- [x] 클릭 → 툴팁(관계 정보) → 수정 버튼 → 편집 팝업
- [x] 캐릭터 노드 클릭 → 상세 패널 (인물관계 탭 먼저)

**세계관 탭**
- [x] 폴더/하위폴더 구조, 노트 작성
- [x] 정렬, 브레드크럼 네비게이션
- [x] 우측 상세 패널 노트 편집

**용어집 탭**
- [x] 용어 + 설명 입력폼, 목록
- [x] 핀(📌) on/off 토글
- [x] 집필 화면 연동 (핀 고정 용어 하이라이트)

### 공통 UX
- [x] 전체 목록 삭제 아이콘 (⊖, hover 시 표시)
- [x] 커스텀 확인/취소 모달 (window.confirm 대체)
- [x] 작품별 통계 바 (챕터, 캐릭터, 글자 수, 세계관 노트, 수정일)
- [x] 섹션 헤더 패턴 통일 (좌 타이틀 / 우 뱃지 / 힌트 문구)

## 파일 구조

```
src/
├── App.tsx                          # 라우팅 + Provider
├── index.css                        # CSS 변수 + 다크모드
├── contexts/
│   ├── AuthContext.tsx               # 인증
│   ├── ThemeContext.tsx              # 다크모드
│   └── ConfirmContext.tsx + .css     # 커스텀 confirm 모달
├── pages/
│   ├── Dashboard.tsx/css
│   ├── MyWorks.tsx/css
│   ├── WorkDetail.tsx/css            # 메인 작품 화면 (6개 탭)
│   ├── NewWork.tsx/css
│   ├── Landing.tsx/css
│   ├── Login.tsx / Signup.tsx / Auth.css
│   └── Profile.tsx/css
├── components/
│   ├── CharacterDetail.tsx/css       # 캐릭터 상세 오버레이
│   ├── RelationDiagram.tsx/css       # 관계도 SVG
│   ├── Header.tsx/css
│   └── Footer.tsx/css
└── data/
    └── sampleData.ts                # 타입 + 샘플 데이터
```
